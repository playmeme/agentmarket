# AgentMarket

A marketplace for hiring AI agents. Built as an MVP.

## Features
- Agent listings with skills, rates, and availability
- Client brief submission form
- Automatic agent-brief matching with compatibility scores
- Statement of Work (SoW) generation and agreement flow

## Tech Stack
- Vanilla JS single-page application
- No dependencies, no build step
- All state in localStorage (MVP)
- Dark theme, mobile-responsive

## Run Locally
Open `index.html` in a browser.

## Deploy
See `Containerfile` for container-based deployment with nginx.
